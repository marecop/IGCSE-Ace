#  IGCSE ACE — AI-Powered Practice & Exam Preparation Platform

**IGCSE ACE** is an intelligent learning platform designed specifically for IGCSE students.  
Powered by advanced AI question generation and automatic marking, it helps students identify weaknesses, strengthen core knowledge, and effectively prepare for real examinations.

---

##  Key Features

- **AI-Generated Questions**
  Automatically produces high-quality questions based on difficulty level, topic, and exam board requirements.

- **AI Marking & Detailed Feedback**
  Instant grading with explanations, helping students understand mistakes and improve rapidly.

- **Two Study Modes**
  - **Practice Mode:** Train question by question, focusing on specific topics.
  - **Mock Exam Mode:** Full exam simulation with all required tasks in one session, replicating real exam pressure and timing.

### **Cambridge (CAIE / CIE)**
